# Expense Tracking System

This is a simple Expense Tracking System application.

## Features

* Add and manage expenses
* View expense reports
* View graphical repesentation of your expenses and incomes.
* Export expense data

## Installation

1. Clone this repository.
2. Install dependencies using `npm install`.
3. Configure environment variables.
4. Run the application using `npm start`.

## Usage

1. Add your incomes.
2. Add your expenses.
3. View reports to track your spending.
